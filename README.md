# Career Navigator AI — Landing Page

AI-powered career guidance and job search platform. Built with React, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy to GitHub Pages

```bash
npm run build
npm run deploy
```

## Tech Stack

React 18 • Vite • Tailwind CSS 3 • Framer Motion • React Router 6 • Lucide Icons • React CountUp
